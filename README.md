# NovaCraft Studio Website

Premium multi-page agency website built with **HTML + TailwindCSS + Vanilla JavaScript**.

## Project Structure

```
.
├── index.html
├── services.html
├── pricing.html
├── portfolio.html
├── about.html
├── contact.html
└── assets
    ├── css
    │   └── styles.css
    ├── js
    │   └── main.js
    └── images
```

## Features

- Fully responsive, mobile-first layout
- Sticky modern navbar + mobile menu
- Premium gradients, glassmorphism cards, soft shadows
- Smooth reveal animations on scroll
- SEO-friendly meta tags
- Pricing tiers with highlighted "Most Popular" plan

## Deployment

Deploy directly to **Netlify** or **Vercel** as a static site.
